:mod:`menpofit.io`
==================

Menpofit includes the ability to save and load pre-trained models for
specific tasks. This module contains code for
pickling down, downloading, and restoring fitters efficiently.

If you make use of one of menpofit's pre-trained models, you will find that
the type that is provided to you is the ``PickleWrappedFitter``. See it's
documentation to understand it's purpose and how you can effectively use it.
